Request a document

Use this service to ask us to add a document to a company's filing history if it is not currently available.

It takes up to 2 working hours for us to check the availability of a document and add it to a company's filing history between 8.30am - 4pm, Monday - Friday (excluding holidays).

If you make a request outside of this time, the document will be added the next working day.

Check the company filing history for updates on the status of the document when you have made the request.

Requesting a document takes around 5 minutes.
